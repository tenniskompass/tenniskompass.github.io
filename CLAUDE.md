# TennisKompass – Webseite

## Tech Stack
- Next.js 15 + TypeScript + Tailwind CSS
- Deployment: GitHub Pages (statischer Export)

## Projektbeschreibung
Ratgeber-Blog für Tennis-Eltern in Deutschland.
Domain: tenniskompass.de

## Design-Regeln
- Nutze das AskUserQuestion Tool, um den Nutzer über das Websitedesign zu interviewen
- Nutze UI/UX Pro Max für Design-System-Generierung
- Nutze den Frontend-design Skill für alle UI-Entscheidungen
- Keine generischen AI-Aesthetics
- Performance-optimiert (Core Web Vitals)
- Nutze SEO-Audit
